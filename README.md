# cm3d2

Custom Maid 3D2

## Environment

- OS:
    - Windows 10
    - Ubuntu 18.04
- Node.js: 12.14.1
    - Yarn package manager: 1.21.1

### Setup TypeScript
```bash
# install node_modules
## $ yarn init -y
## $ yarn add typescript @types/node ts-node
$ yarn install

# execute test code
$ yarn ts-node test.ts
```
